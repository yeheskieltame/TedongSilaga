COMMAND FOR AI AGENT
ADVANCED UI/UX DESIGN — TEDONG SILAGA PREDICTION MARKET
1. Objective

Design a high-quality Web3 prediction market UI/UX for a platform called:

Tedong Silaga Arena

The platform allows users to predict the winner of traditional Toraja buffalo matches.

Users place predictions on:

Buffalo A
Buffalo B

Users stake tokens:

WLD
USDC

The platform runs on:

World Chain

Users must verify identity using:

World ID

The design must combine:

Morpho-level visual design
Polymarket-style prediction UX
Game retention mechanics
Cultural Toraja identity
2. Key UX Goals

The UI/UX must achieve these goals:

1. Users understand the game instantly
2. Users place prediction within 5–10 seconds
3. Users frequently return to the platform
4. Users feel competition and excitement
5. Platform looks trustworthy and premium

The design should NOT feel like gambling.

It should feel like:

prediction market
strategy game
Web3 protocol
3. UX Inspiration

Focus on:

Focus on:

4. Core Game Loop (Retention Design)

The platform must follow this game loop:

User opens platform
↓
Sees active matches
↓
Predicts a winner
↓
Waits for result
↓
Wins or loses
↓
Claims reward
↓
Sees next upcoming matches
↓
Plays again

The UI must encourage continuous participation.

5. Retention Mechanics (Important)

Design UX that encourages users to return.

Include features such as:

Streak System

Example:

Winning Streak: 4 matches

Display streak badges.

Leaderboard Competition

Users see rankings:

Top Predictors Today
Top Predictors This Week
Top All-Time Players

Competitive UI increases engagement.

Upcoming Matches

Show upcoming events prominently.

Example:

Next match starts in 20 minutes

Countdown timer encourages users to stay.

Activity Feed

Display:

Recent Predictions
Recent Winners

This creates social proof.
